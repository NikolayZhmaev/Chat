Работа с сетью
Продолжаем развивать чат!
 1. Ограничить доступ, т.е. под одним логином может подключаться только однин клиент.
 2. Реализовать личные сообщения, если клиент пишет «/w nick3 Привет», то только клиенту с ником nick3 должно прийти
    сообщение «Привет».
 3. Решить проблемму с завершением работы потока при закрытии окна клинта.
 4. Реализовать "черный список" без использованием БД
 5. Реализовать окошко со списком подключенных к чату клиентов.
